This component represents a list.

Several types of list are supported, including a definition list (also known as a describe list).

The following may be specified as an attribute on a template element:

  • list-items: The content to display in a section within the list representing all items of the list.

The following attributes are available:

  • list-type: One of described, group-custom-content, group-flush, group-div, group-ui, ordered, unstyled, or unordered.
