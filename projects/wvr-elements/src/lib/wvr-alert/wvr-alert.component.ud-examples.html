This component represents an alert.

An alert may be closed automatically, on event, or not closable at all.

The following attributes are available:

  • alert-type: One of basic, self-closing, or custom.
  • closeable: One of true or false.
  • closeTimer: A unit of time in milliseconds for how long to wait before auto-closing.
 The basic wvre-alert component with no customizations. The wvre-alert component with self closing feature based on the input closing timer. The wvre-alert component with custom content.
