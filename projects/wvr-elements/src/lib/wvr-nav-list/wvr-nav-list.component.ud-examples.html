This component represents a navigation list.

A navigation list is a special type of list fine-tuned for being used in navigating a page.

The following may be specified as an attribute on a template element:

  • list-items: The content to display in a section within the list representing all items of the list.

The following attributes are available:

  • aligned: One of center, left, right, or any valid CSS alignment value.
  • vertical: One of true or false.
 The wvre-nav-list component has the weaver-nav-li as a sub component.
